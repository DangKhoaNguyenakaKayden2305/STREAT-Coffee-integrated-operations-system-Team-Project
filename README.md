**Solution Overview**

We designed an Integrated Operations Management System (IOMS) that:

Preserves existing systems (Square, Xero, Shopify, FoodStorm)

Introduces a central data integration layer

Automates inventory, costing, and reporting workflows

Provides role-based dashboards for different stakeholders

Key Solution Components

Centralised data hub (real-time synchronisation)

Purchase Order management workflow

Real-time inventory tracking across locations

Automated recipe costing with margin alerts

Operational & financial analytics dashboards

**Power BI Dashboards (Decision-Focused)**

The dashboards were designed to support real business decisions, not just reporting.

Examples of insights provided:

Daily sales vs inventory movement by location

Live gross margin tracking by recipe

Cost variance alerts (>25% threshold)

Cross-location performance comparison

End-of-day reconciliation (replacing 6-week delays)

These dashboards enabled next-day action, instead of retrospective analysis.

**Business Process Modelling (BPMN)**

BPMN diagrams were used to:

Visualise current vs future-state processes

Reduce ambiguity between business and technical teams

Ensure workflows matched how STREAT actually operates

Support scalability and future system implementation

The BPMN artefacts became a shared language across the team.
